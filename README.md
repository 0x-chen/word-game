这是一个英语单词游戏，玩家需要在选项中选择正确的中文翻译，帮助青蛙跳到正确的位置。游戏包含多种模式和难度级别，界面友好，适合练习英语单词。

## 安装和运行

1. 克隆仓库：
   ```bash
   git clone https://gitee.com/Chen-Lewen688/word-game
   ```

2. 确保已安装 Python 3，用于处理单词列表：
   ```bash
   python3 --version
   ```

3. 将项目文件放在本地服务器中运行，例如使用 Python 的简易服务器：
   ```bash
   cd word-game
   python3 -m http.server
   ```

4. 打开浏览器，访问 `http://localhost:8000` 运行游戏。

## 使用说明

1. 打开游戏后，选择游戏模式（关卡模式或无尽模式）。
2. 在关卡模式中，选择一个单元和难度级别开始游戏。
3. 游戏中，选择与显示英文单词匹配的中文翻译。
4. 如果选择正确，青蛙会跳到正确的位置；如果错误，青蛙会掉入水中，游戏结束。
5. 游戏结束后，可以查看得分和正确回答的单词数量，并选择重新开始或返回主菜单。
