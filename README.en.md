This is an English word game where players must select the correct Chinese translation from the given options to help the frog jump to the right position. The game features multiple modes and difficulty levels, with a user-friendly interface ideal for practicing English vocabulary.

## Installation and Running

1. Clone the repository:
   ```bash
   git clone https://gitee.com/Chen-Lewen688/word-game
   ```

2. Ensure Python 3 is installed for processing the word list:
   ```bash
   python3 --version
   ```

3. Serve the project files locally using a simple Python server:
   ```bash
   cd word-game
   python3 -m http.server
   ```

4. Open your browser and navigate to `http://localhost:8000` to play the game.

## How to Play

1. After launching the game, select a game mode (Level Mode or Endless Mode).
2. In Level Mode, choose a unit and difficulty level to begin.
3. During gameplay, select the Chinese translation that matches the displayed English word.
4. If correct, the frog jumps to the right position; if incorrect, the frog falls into the water and the game ends.
5. After the game ends, view your score and the number of correctly answered words, then choose to restart or return to the main menu.
